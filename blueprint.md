
# Lotto Number Generator

## Overview

A simple web application to generate lottery numbers.

## Project Outline

*   **`index.html`**: The main HTML file.
*   **`style.css`**: The CSS file for styling.
*   **`main.js`**: The JavaScript file for generating lottery numbers.

## Current Plan

1.  **Modify `index.html`**: Add a title, a button to generate numbers, and a place to display the numbers.
2.  **Modify `style.css`**: Add styling to make the page visually appealing.
3.  **Modify `main.js`**: Write the JavaScript code to generate the lottery numbers and display them on the page.
